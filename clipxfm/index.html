ClipXfm

AI-Powered Clipboard Transformer

ClipXfm is a lightweight, blazingly fast desktop utility that automatically monitors your clipboard. Whenever you copy text, a small floating toolbar instantly appears near your mouse cursor. Click a button to run AI tasks like summarization, translation, or grammar check, and copy the results back in a single action.

✨ Key Features

  • Automatic Interception: Runs quietly in your system tray and shows a toolbar only when text is copied.
  • Hybrid AI Engine: Stream answers from cloud providers (Google Gemini, OpenAI, Anthropic) or connect to offline local LLMs via Ollama or LM Studio.
  • Custom Action Buttons: Edit a simple configuration file to define up to 6 custom buttons, icons, and prompts.
  • Secure Storage: API keys are securely saved locally using the operating system's secure credential store.
  • Prompt Injection Defense: Employs XML encapsulation, role segregation, and prompt guards to protect your workstation.

🚀 Quick Start & Usage

  1. Copy any text from any application (browser, PDF, document editor).
  2. The ClipXfm toolbar will slide open near your mouse pointer.
  3. Click a command button (e.g. Summarize, Translate) or enter custom instructions.
  4. The AI streams the result in real-time. Click Copy & Close to copy the output back into your clipboard.

⚙️ Customizing Commands

You can define custom prompts and shortcuts by creating a commands.json file in your app's Data Directory (open settings from the tray icon and click "Open Commands Directory" to locate it). 

[
  {
    "label": "Bullet Points",
    "icon": "list",
    "systemPrompt": "Convert the text into a concise bullet point list. Output in \${lang}."
  },
  {
    "label": "Translate",
    "icon": "languages",
    "systemPrompt": "Translate the text into \${lang}. Provide ONLY the translation."
  }
]

The Target Language specified in your UI settings automatically replaces the \${lang} variable.

🛡️ Windows SmartScreen Warning

As the application installer is unsigned, Windows SmartScreen will block execution on first launch. You can securely bypass this warning:

  • In the blue "Windows protected your PC" prompt, click the "More info" link under the text.
  • Click the "Run anyway" button that appears in the bottom right corner of the window.
